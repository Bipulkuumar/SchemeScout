# 🔍 SchemeScout

**Find Government Schemes You Actually Qualify For**

SchemeScout is a web application that helps Indian citizens discover government welfare schemes they are eligible for. It uses a smart matching engine to filter central and state schemes based on your age, gender, location, occupation, income, and social category.

---

## ✨ Features

- **Smart Eligibility Matching** — Answer a 2-minute questionnaire and get personalized scheme results
- **Scheme Discovery** — Browse 500+ central and state government schemes
- **Match % Indicator** — See how closely each scheme matches your profile
- **Detailed Scheme Info** — View benefits, eligibility criteria, and step-by-step application guides
- **Search & Filters** — Filter schemes by category and sort by relevance or deadline
- **Responsive Design** — Fully mobile-friendly, works across all screen sizes
- **Smooth Animations** — Premium page transitions and micro-interactions

## 🚀 Getting Started

### Prerequisites
- Node.js 18 or higher
- npm

### Installation

```bash
# Clone the repo
git clone https://github.com/Bipulkuumar/SchemeScout.git
cd SchemeScout

# Install dependencies
npm install

# Start development server
npm run dev
```

The app will open at `http://localhost:5173/SchemeScout/`

### Build for Production

```bash
npm run build
```
